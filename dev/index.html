Laska.jl documentation

Welcome to the WIP documentation for the WIP package Laska.jl.

Laska.jl is an attempt at providing functions for importing, manipulating and visualizing data obtained from Neuropixel probes using spikeGLX and preprocessed in KiloSort and Phy.
