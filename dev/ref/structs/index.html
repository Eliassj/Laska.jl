Concrete types

"Experiment" wrappers

PhyOutput

Laska.PhyOutput — Type
mutable struct PhyOutput{T} <: AbstractExperiment{T}
    clusterids::Vector{Int64}
    clusters::Vector{Cluster{T}}
    trigtimes::Vector{T}
    meta::Dict{SubString{String},SubString{String}}
end

Struct for holding Kilosort output preprocessed in Phy. Should be instantiated using the outer constructor Laska.importphy.

Direct field access is not recommended. Basic interface functions include:

  • Laska.clusterids – Returns all cluster ID:s as a Vector.
  • Laska.getcluster – Returns a specific Laska.Cluster.
  • Laska.clustervector – Returns all Laska.Cluster:s.
  • Laska.getmeta – Returns the spikeGLX meta as a dict or a specific entry.
  • Laska.triggertimes – Returns the trigger event times.
  • Laska.ntrigs – Returns the length of the trigger event time Vector.

Single cluster wrappers

Cluster

Laska.Cluster — Type
struct Cluster{T} <: AbstractCluster{T}
    id::Int64
    info::Dict{String,String}
    spiketimes::Vector{T}
end

Struct for holding a single Cluster.

Direct field access is not recommended. Basic interface functions include:

  • Laska.id – Returns the Cluster id.
  • Laska.nspikes – Returns the number of spikes in the cluster (Based off length of the spiketimes field).
  • Laska.info – Returns the info of the Cluster from "cluster_info.tsv" as a Dict.
  • Laska.spiketimes – Returns a Vector containing all spiketimes of the Cluster.
