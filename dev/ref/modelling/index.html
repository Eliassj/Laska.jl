Hodgkin-Huxley model

α and β functions

Generalized version

LaskaML.ab_generalized — Function
ab_generalized(V, p)

The generalized form of α and β.

function ab_generalized(V, p)
    A, B, C, D, F, H = p
    (A + B * V) / (C + H * exp((V + D) / F))
end
source

A generalized version of the α/β functions as per Nelson, M.E. (2004)

Calculated as:

\[\alpha(V) = \frac{A + BV}{C+Hexp(\frac{V+D}{F})}\]
